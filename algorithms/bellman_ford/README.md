# Bellman-Ford algorithm

## 要点

- 重み付き有向グラフにおける、**単一視点** の最短経路問題を解くアルゴリズム。
- 負のエッジが存在する場合でも正しく計算でき、負の経路の存在を検出・報告することができる。
- アルゴリズムの正しさは数学的帰納法で示すことができる。

## 計算量

`O(|E| * |V|)`

優先度付きキューを使えば線形対数となるダイクストラ法よりも大きい。
